## 介绍

本软件受utools启发，以tauri框架实现的一款轻量级PC端应用功能集合。

目前已实现的功能有：

- [x] 快速搜索启动本地所有已安装程序
- [x] 快捷打开网页链接
- [x] 本地文件搜索app
- [x] explorer窗口识别、启动应用

未来希望实现的功能：

- [ ] linux、mac系统适配
- [ ] 全局启动面板
- [ ] 图像处理app
- [ ] Markdown编辑器
- [ ] 截图工具
- [ ] ...

## 开发扩展

由于开发一套插件系统过于复杂繁琐，个人精力有限，且目前没有比较好的思路，所以暂时仅支持源码层次的应用扩展。

如果你想要开发扩展，可以直接克隆本项目，更多开发细节可以参考文章：[Ombra开发](https://www.kucoding.com/article/222.html)
